# Kannada-Movies-2025-AI-Analytics
An end-to-end Data Analytics project that analyzes the performance of notable Kannada movies released in 2025 using AI, Excel, SQL, and Power BI.

## Project Overview
This project explores the Kannada film industry's 2025 releases by analyzing budgets, box office collections, profitability, ROI, IMDb ratings, genres, directors, actors, production houses, and OTT platforms. The goal is to convert raw movie data into meaningful business insights and interactive dashboards that support data-driven decision-making.

## Objectives
* Analyze the commercial performance of notable Kannada movies released in 2025.
* Identify the highest-grossing and most profitable movies.
* Compare Return on Investment (ROI) across films.
* Study trends based on genres, directors, actors, and production houses.
* Build interactive dashboards for business intelligence.
* Generate AI-assisted insights and strategic recommendations.
